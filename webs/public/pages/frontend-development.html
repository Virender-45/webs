Frontend Development

Crafting responsive, accessible, and visually polished user interfaces using modern web standards — HTML, CSS, Bootstrap & JavaScript.

Overview

Building the Web, Pixel by Pixel

Frontend development is where design meets engineering. I focus on translating wireframes and ideas into live, interactive experiences — making sure every element is precisely placed, every interaction feels smooth, and every layout works from a 320px phone to a 4K monitor.

My approach prioritises semantic HTML for accessibility, clean modular CSS, and progressive enhancement so the core experience works everywhere.
