Skill Deep-Dive

Responsive Web Design

Creating layouts that work beautifully everywhere — from a 320px phone to a wide-screen desktop — using Flexbox, CSS Grid, and Bootstrap breakpoints.

Philosophy

Mobile-First, Then Scale Up

Starting from the smallest screen forces you to prioritise what actually matters. Once the mobile layout is solid, progressively enhancing for tablet and desktop is straightforward and avoids the common mistake of cramming a desktop layout onto small screens.

This portfolio itself is fully responsive — resize your browser to see Flexbox and CSS Grid adapting in real time.

Fluid Grid Pattern

/* Auto-responsive grid — no breakpoints needed */ .card-grid { display: grid; grid-template-columns: repeat(auto-fit, minmax(260px, 1fr)); gap: 24px; } /* Flexbox nav that collapses gracefully */ .nav { display: flex; gap: 32px; flex-wrap: wrap; } @media (max-width: 768px) { .nav { flex-direction: column; position: fixed; /* Drawer */ } }
